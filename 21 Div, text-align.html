Todos los tags dentro de un div pueden ser editados sin necesidad de poner ningún class o id en cada uno.

Se pueden seguir poniendo class o id por separado. Por ejemplo, aquí le he cambiado el color a este párrafo. Al hacerlo, he hecho lo que se le llama "inheritage" (sobreponer el color del párrafo frente al del div).

En caso de que sean más de un div, lo diferenciamos de los demás div con un id.

Los div pueden ser útiles para editar varios tags y ordenar la página web en secciones.
