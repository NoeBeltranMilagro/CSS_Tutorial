La property "float" posicionará el elemento en el lado que especifiquemos.

En este ejemplo, hemos puesto un float en cada tag para que cada uno esté en un extremo.

Se puede ajustar el tamaño que ocupe el bloque del texto con la property width.

En este caso, hemos hecho lo mismo que en el primer ejemplo, pero utilizando un width para ocupar el 50% de la página y que queden perfectamente alineados.

Si le ponemos a ambos un float a la izquierda,

Rellenerán el espacio restante en el centro automáticamente.
